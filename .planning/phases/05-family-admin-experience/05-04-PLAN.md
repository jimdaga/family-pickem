<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser POST -> invite commands | Untrusted invite IDs, roles, expiries, and max-use values request invite mutations. |
| invite creation -> browser display | Raw invite code leaves the server once for copy/paste. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-05-09 | Information Disclosure | invite list/template/audit | high | mitigate | Never render or log raw code/hash after immediate creation; test non-disclosure. |
| T-05-10 | Elevation of Privilege | invite role creation | high | mitigate | Allowlist invite roles by actor role; admins cannot create elevated invites unless owner rules allow it. |
| T-05-11 | Tampering | revoke/regenerate | medium | mitigate | Scope invitation lookup to current family and wrap revoke/create in transaction. |
| T-05-SC | Tampering | npm/pip/cargo installs | high | mitigate | No package installs are planned; if introduced, stop for package legitimacy review. |
</threat_model>
